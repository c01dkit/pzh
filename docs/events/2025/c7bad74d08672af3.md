# Princeton Puzzle Hunt
- 主题：Princeton Puzzle Hunt 2025
- 时间：2025-05-03 13:30 ～ 2025-05-03 17:30
- 主办方：Princeton University
- 网站链接：[https://puzzles.princeton.edu/hunt/2/view/](https://puzzles.princeton.edu/hunt/2/view/)

由普林斯顿大学举办的puzzlehunt，2025年版本为单日4小时活动，共16道题，分为3个常规分区与1个最终meta分区，每个常规分区以一道meta题收尾。每队最多4人，所有题目答案均为英文单词或短语。比赛期间每10分钟自动释出一条提示，使用提示不扣分。由Jane Street、Projects Board赞助，基于PuzzleSpring平台。时间为EDT。
