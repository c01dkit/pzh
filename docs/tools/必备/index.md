# 必备

共 4 个工具。

## Nutrimatic

- 链接：<https://nutrimatic.org/2024/>
- 关键字：查词

词组正则匹配网站

## 千篇国学

- 链接：<https://www.qianp.com/>
- 关键字：成语, 字典, 词典

千篇国学，右上角可以选字典、词典、成语，支持定字查找，非常好用

## 字统网

- 链接：<https://zi.tools/>
- 关键字：汉字

支持部件 结构

## 搜韵

- 链接：<https://sou-yun.cn/>
- 关键字：古诗词

支持诗词查询，可按作者、朝代、体裁、韵部、关键字词及其位置精确查询，也可像百度、谷歌一类模糊检索。可查典故、词谱、诗词地图、对仗词汇。支持定字查询。
