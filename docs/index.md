# Puzzlehunt 知识库
 
这是一个用于收录与浏览各类 **puzzlehunt（解谜比赛）** 的 MkDocs 站点。

## 什么是Puzzle？

??? abstract "ZJU Puzzle Hunt 的介绍"
    维基百科对于谜题（Puzzle）的定义是：谜题是可以测试人类心智或掌握知识的一类游戏、问题或玩具。

    你之前肯定或多或少接触过一些谜题，试卷上的数学题，报纸上的填字游戏，推理小说中的密码，电子游戏中的推箱子等等，广义上说，这些都属于谜题的范畴。

## 什么是Puzzle Hunt？

??? abstract "SECO2 的介绍"
    你可能已经接触过各种各样的谜题：杂志上的数独、经典的逻辑推理题，语文课上的灯谜，英语课上的拼图游戏。也许你和同学玩过“你画我猜”，曾用摩尔斯电码加密信息让朋友破译；研究过围棋的棋局，读过那些引人入胜的悬疑小说。或者，你的手机游戏列表里有《纪念碑谷》这样的空间解谜游戏，又或者在周末参加过户外的寻宝活动。

    在 Puzzle Hunt（解谜寻宝）中，你和你的团队需要解决的不止是一道，而是一系列这样的谜题，并从中发掘隐藏的线索。

    想象一下，任何事物都可以成为谜题：看似是拼图游戏，实际上是对拼图碎片形状的推理；或是音符的排列，每组音符代表一种乐器。奇妙的谜语，严密的逻辑，隐藏的符号，甚至可能是一段代码中变量的命名规律，或是一部科幻电影里隐藏的彩蛋，万事万物皆可成谜。
    
    而你的目标，就是找到合适的线索，选择最符合当前情况的方法，从里面找到你的答案。无论如何，最终你一定能得到一组有意义的词语或一句话。

    在解决一连串的谜题之后，元谜题（Meta Puzzle）会将你之前得到的所有答案串联成新的谜题，形成错综复杂的结构。

    最终，Puzzle Hunt 的目标是团队共同解决所有出现的元谜题（是的！你不需要所有小题的答案就可以解决元谜题！），得到隐藏在谜题和谜题结构中的最终答案（通常被称为「Final Meta」）。

    当队伍里有超过一个人时，我们强烈建议你准备一个共享表格，实时记录每题的题目、答案、解题过程和想法，并与队友共享。这不仅能帮助团队成员更好地分工协作，还能让每个人都能随时了解解题进度，从而提高解题效率。

??? abstract "ZJU Puzzle Hunt 的介绍"
    Puzzle Hunt 通常需要参与者通过团队合作解开一系列的谜题。这类谜题往往更不依赖特定方向的预备知识，谜题设计手段也更为多样。逻辑谜题、古诗词、文学、影视、游戏、音乐、语言学、密室逃脱……你的兴趣爱好可能就是某个 Puzzle Hunt 中的下一道题！

    除此之外，Puzzle Hunt 更为典型的特征是它的结构。Puzzle Hunt 常常分为多个区域（Round），在每一个区域中，除了若干普通谜题外，往往还有一个元谜题（Meta），Meta 一般会在解谜者解决了该区域的大多数普通谜题后解锁，并且通常需要解谜者利用该区域的普通谜题的答案来解答。

    随着 Puzzle Hunt 的发展，结构的设计变得愈发灵活。部分谜题可能会对整个活动进行更全面的回顾与挖掘，比如需要你用到其他谜题的答案、甚至剧情或者某个美工元素。这种安排使得谜题之间以某种特定的联系被组织在一起，从而给解谜者一种高层次的美感。
