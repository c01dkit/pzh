# Trivia / Research（检索与知识）

本页用于沉淀 **Trivia / Research（检索与知识）** 这一题型的通用解法与可复用技巧。

## 题型特征（你可以逐条补充）

- 常见题面信号：例如关键词、格式、异常之处、可疑的分隔符/排版等
- 常见变换：替换/重排/分组/映射/同音/首字母等
- 常见提取：索引取字、排序后取位、坐标系/网格取字等

## 常用解题流程（建议模板）

1. 观察：列出可疑之处（长度、重复、分隔、标题、主题词）
2. 假设：提出 2–3 种可能机制并快速验证
3. 落地：把机制写成可重复操作（必要时写脚本/做表）
4. 提取：确认提取方式与答案形式（通常是英文单词/短语）
