## 题目详情

!!! abstract ""
    *万国统一的第二平面上飘着好多不认识的字啊，要怎么让他们落到地上呢*

注意看，这个人坐在老式飞机上飞行，而他竟然来到了朝韩两国的共管地带，怕不是落地就要变成某种无机材料了。（十九）

这座墓穴的上方早已长草，把它挖开之后，其下方竟然没有棺材，只有放着刀的石臼，或许墓主已经经历了某件事？（十六）

在高大的石墙东侧是一片森林，而即便能绕过石墙，你也会发现森林南侧是一块告示牌，作为景区来说，这怎么能让我露出某种感情呢？（十八）

一代人垮掉了，他们变成了某种恶心的动物，并且公式一样地躲在保护结构之下，唉，这就是被我们倾注某种东西的下一代啊！（九）

这只大眼睛在看什么呢？原来在看一个罐头，只不过不知为什么罐头左上角的拉环丢了，让眼睛的主人心急如焚。其实这个拉环就在眼皮子上方，为什么他像某种东西那样呆呆地定在那里，不多看一眼呢？（十一）

这张毛巾挂在倾斜的十字挂钩上，已经摇摇欲坠。在它的旁边，一只虫子宣告着此处的所有权，更彰显了无人之处，某种东西的影响力。（十一）

苦命的方形小岛被分割成了叠压的三个部分，但总有人没有忘记叠在上面的使命——回归吧！这种分裂还要持续到什么时候？即便相差九十度，也是可以有共同点的啊！（二十三）

一只猪的头上竟然长出了两个角，这让它只能孤零零地待在草屋之下。不过富有戏剧性的是，这个草屋也有一对翅膀，和猪的两个角相映成趣——每个人都有不合群的体质，让我们学会接纳对方，就算我们相互平行。（十四）

这是一艘残破的船，它的读音告诉了我们它原本该有的样子，但船底和船夫都不知所踪，整艘船只剩一个顶和一个空架子，让人感到强烈的反差，就像某种特定的图案一样。（十三）

草地初生，纤细柔软，一切本来应该是早春的欣欣向荣，直到一个刺眼的钓钩出现在不起眼的角落，就像在游戏厅爽玩之后看到的某个人一样，让人胆战心惊。（十）

## 总结

??? note "题解"
    ft中的**万国统一**提示万国码，即Unicode。google得到这个[unicode网站](https://symbl.cc/en/unicode-table/#cjk-unified-ideographs-extension-b)，注意到正好有个2: Supplementary Ideographic Plane可以和ft中的**第二平面**对应。

    根据提示2（或者解字谜“朝韩两国的共管地带”-> 𠦝），以及“老式飞机”->飞（飛）放到字统网，利用搜字功能得到“𩙶”，其包括了题目中的“人”、“𠦝”、“老式飞机（飛）”等构件，提交验证得知是小题答案之一。其Unicode编码为0x29676，根据提示3或4，取低2字节，得到0x9676，对应汉字“陶”，和小题最后半句话“某种无机材料”对应。

    根据以上思路进行解题，得到：

    | 搜字组件 | 结果 | Unicode编码 | 去除高位 | 对应结果 | 备注 |
    | --- | --- | --- | --- | --- | --- |
    | 𠦝飞 | 𩙶 | U+29676 | 0x9676 | 陶 | |
    | 艹刀臼 | 𦽜 | U+26F5C | 0x6f5c | 潜 | |
    | 石林示 | 𥖜 | U+2559C | 0x559c | 喜 | |
    | 虫弋 | 𧈱 | U+27231 | 0x7231 | 爱 | |
    | 目丿干凵 | 𦤍 | U+2690D | 0x690d | 植 | 把“罐头”的“缶”拆开了，实际上应该是“目丿𠙻” |
    | 巾虫 | 𧉩 | U+27269 | 0x7269 | 物 | 一个干扰项是“䖷”，不过它既不是第二平面，“十字”也不够“倾斜” |
    | 归目 | 𥍁 | U+25341 | 0x5341 | 十 | 叠在一起的三个方形小岛是“目” |
    | 冫㒸 | 𤸉 | U+24E09 | 0x4e09 | 三 | “㒸”比较好想，“草屋”对应“广”属实有点抽象了，不过用“翅膀”、“相映成趣”倒也勉强能搜出来 |
    | 竹夆 | 𥭗 | U+25B57 | 0x5b57 | 字 | 何意味，完全没关系好吧！？ |
    |  | 𦯍 | U+26BCD | 0x6bcd | 母 | 搜不出来，可以用“母”加上0x20000来反爆 |

    得到`陶潜喜爱植物十三字母`，即菊花，CHRYSANTHEMUM。
