??? note "题解"
    礼物列表其实暗示了各个小题的主题。

    ??? note "P·Index送的礼物"
        根据提示1，题面来自[伊索寓言](https://en.wikipedia.org/wiki/Perry_Index)，给出的三个式子能和序号对应上。检索一下题目最后列出的词组，能够和P-Index对上的包括：
        
        | 题目列表 | P-Index | 备注 |
        | --- | --- | --- | 
        | A BLIND MAN | 37 |  |
        | TWO FROGS | 43 |  |
        | THE HALCYON | 25 |  |
        | THE ENEMIES | 68 |  |

        正好四个，相加得到173，对应P-Index中的Hermes and the Woodcutter。这个故事有个别名[Mercury And The Woodman](https://read.gov/aesop/102.html)可以和题目给出的7 3 3 7对应上。

    ??? note "传送带上的礼物"
        根据提示6，从题目列表中寻找单词长度为7的词，并将其填充到矩阵里，使得对角线为单词PRESENT
        ```
        P	H	O	E	N	I	X
        W	R	I	N	K	L	E
        U	S	E	T	O	O	L
        O	N	E	S	E	L	F
        P	A	C	H	E	C	O
        F	I	R	S	T	N	L
        C	H	A	R	I	O	T
        ```
        然后按照传送带的顺序读取结果`HITECLOTHESW`，谷歌一下获得`WHITE CLOTHES`
