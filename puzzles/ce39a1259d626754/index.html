    题解

    由于整个欣年3比赛都和马年有关，所以直接提交PONYTA

    根据提示1和2，先拿到宝可梦图鉴

    先解meta3。猜想每行计算结果A1Z26以后是宝可梦，将sum(O)、sum(T)、sum(D)视作3个变量，根据计算结果都是1-26的整数且是宝可梦名字来编写solver，得到唯一解GASTLY。

    然后再解meta1，但看不懂。此时注意到meta5的黄色和绿色很像wordle，且meta3的六字母恰好和wordle长度一致。结合内循环meta不妨大胆猜想整个4行6列的格点表示meta1-meta4的答案。那么一致meta5的答案不存在gastly字母，且meta1的答案是黄黄绿黄白白，直接爆破查找可行的meta5和meta1的答案。发现在所有可行解（dodrio、bidoof、pidove）中，无论meta5是哪一个，meta1始终为oddish。所以meta1为ODDISH。

    然后观察meta2，注意到“答案第二字去掉部首可与圈中字组词”。在已知meta1和3种meta5的情况下，meta2的取值是有限的。能和字母组词的也就buizel泳圈鼬的圈去掉口，变成卷，可以和A组词：A卷；或者kyogre盖欧卡的欧去掉欠，变成区，可以和A组词：A区。

    在meta2有限的情况下，直接对meta4进行爆破求解，结果也是有限的（缩小到9个宝可梦，在20次提交以内了，直接爆破）。具体可以运行解题代码。

    还有一些信息没有用到，比如meta3算出来o、d、t的数量是5、4、2，以及meta1的数字不知道干嘛的，以及meta2的“字数相同”和“格式相同”也不知道大干嘛的。

def meta3():

    O, D, T = Ints('O D T')

    s = Solver()

    # T/2 必须是整数，所以 T 必须是偶数
    s.add(T % 2 == 0)

    # 定义6个表达式
    e1 = O + T
    e2 = T / 2
    e3 = O * D - T / 2
    e4 = O * D
    e5 = D * D - D
    e6 = O * O

    # 每个表达式的结果在 [1, 26] 之间
    for e in [e1, e2, e3, e4, e5, e6]:
        s.add(e >= 1, e <= 26)

    # 枚举所有解
    results = []
    while s.check() == sat:
        m = s.model()
        o_val = m[O].as_long()
        d_val = m[D].as_long()
        t_val = m[T].as_long()
        results.append((o_val, d_val, t_val))
        # 排除当前解，继续搜索
        s.add(Or(O != o_val, D != d_val, T != t_val))

    # 输出结果
    print(f"共找到 {len(results)} 组解\n")
    string_result = []
    for o_val, d_val, t_val in sorted(results):
        t_half = t_val // 2
        od = o_val * d_val

        name = ''.join(chr(val + ord('a') - 1) for val in [o_val + t_val, t_half, od - t_half, od, d_val**2 - d_val, o_val**2])
        string_result.append(name)
        if name in pokemon_names:
            print(f"Found a match: {name}")
            print(f"O={o_val}, D={d_val}, T={t_val}")
            print(f"  ① O+T      = {o_val + t_val}")
            print(f"  ② T/2      = {t_half}")
            print(f"  ③ O*D-T/2  = {od - t_half}")
            print(f"  ④ O*D      = {od}")
            print(f"  ⑤ D²-D     = {d_val**2 - d_val}")
            print(f"  ⑥ O²       = {o_val**2}")
            print()


def single_wordle(test, answer):
    result = ''
    for i in range(len(test)):
        if test[i] == answer[i]:
            result += 'G'
        elif test[i] in answer:
            result += 'Y'
        else:
            result += 'W'
    return result
